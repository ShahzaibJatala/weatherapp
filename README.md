# ☁️ Weather Forecast Application

A sleek, real-time weather application that provides accurate weather data for cities worldwide. This project demonstrates advanced **JavaScript** proficiency, specifically in managing **asynchronous API calls** and dynamically updating the **DOM** based on external data sources.

## Features

* **Real-Time Data:** Integrates with the **OpenWeatherMap API** to provide current temperature, humidity, and wind speed.
* **Location-Based Search:** Allows users to search for weather conditions in any city across the globe.
* **Dynamic Imagery:** Automatically updates weather icons and background elements based on the current weather condition (Clear, Clouds, Rain, etc.).
* **Unit Conversion:** Displays temperature in Celsius with high precision.
* **Error Management:** Includes validation to handle "City Not Found" scenarios gracefully.
